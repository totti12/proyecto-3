## Operadores
### Operadores Aritméticos
#### **Operador:** +

**Tipo:** Aritmético	

**Interfaz:** SQL 

**Implementación:** (C)

**Synopsis:**	
```
     Intervalo  +  Intervalo
```

**Descripción:** + Define la operación aritmética adición de intervalos

**Ejemplo:** 
```sql
      select intervalo(e_inc_num) + intervalo(e_mort_tbhiv_num_hi) as sumaIntervalo from tuberculosis
```
