### Operadores Aritméticos
#### **Operador:** +

**Tipo:** Aritmético	

**Interfaz:** SQL 

**Implementación:** (C)

```
     Intervalo  +  Intervalo
```

**Descripción:** + Define la operación aritmética adición de intervalos

**Ejemplo:** select intervalo (diastolic) + intervalo (systolic) from medical
